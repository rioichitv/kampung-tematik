# kampung-tematik
Kampung tematik adalah kawasan permukiman yang dikembangkan dengan identitas atau tema khusus (misalnya budaya, kerajinan, kuliner, atau warna) untuk meningkatkan kesejahteraan masyarakat, kualitas lingkungan, dan menjadi destinasi wisata, dengan melibatkan partisipasi aktif warga dalam memanfaatkan potensi lokal dan memecahkan masalah bersama
